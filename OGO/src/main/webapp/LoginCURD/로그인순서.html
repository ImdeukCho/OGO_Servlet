1. 로그인 팝업 출력 loginTest.html => login.html 2. 로그인 처리 로그인 css 스타일 : loginstyle.css 로그인 되면 메인 페이지로 : login.html => loginServlet => MainTest.jsp 로그안 안되면 로그인 페이지로 : loginServlet => login.html 3. 관리자 로그인 시 메인페이지에서 회원관리메뉴 링크 출력: MainTest.jsp => managementMember.jsp 4. 회원가입 로그인화면 회원가입 클릭 : login.html => createMemberServlet => createMember.jsp 회원가입 시 : createMember.jsp => MemberAddServlet => MainTest.jsp 아이디 중복체크 : idCheck1.jsp => MemberIdCheckServlet => idCheck2.jsp (인클루드 사용) 닉네임 중복체크 : MemberNicknameCheckServlet (ajax 비동기 처리) 5. 회원관리 페이지 회원 삭제 시 : deleteMember.jsp 회원 다중선택 삭제 시 : deleteAllMember.jsp 6. 로그아웃 MainTest.jsp => logoutServlet => loginTest.html
